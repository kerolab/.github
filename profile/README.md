## 研究概要 / Research

アフリカツメガエル（*Xenopus*）などのモデル生物を用い、胚発生（形態形成）の仕組みを通して**生命の進化メカニズム**を解明する。CRISPR-Casなどのゲノム編集技術を駆使し、体のかたちがどのように決まるかを分子レベルで探る。

> We study the evolutionary mechanisms of life by investigating embryonic morphogenesis using model organisms such as *Xenopus* (African clawed frog), employing cutting-edge genome editing tools including CRISPR-Cas.
